Info

  • Each swatch shows how our brand colors map to design tokens and their generic types (i.e. shade, alert, etc).
  • Design tokens should be used if at all possible and the generic types should be used only if a design token doesn't make sense in context.
  • All tokens and types map to tailwind classes (i.e. var(--bg-disabled) => .bg-disable, var(--text-primary) => .text-primary, etc.)
  • The background toggle switches the background color of the page. The dark color toggle shows the different mappings of our dark mode.
